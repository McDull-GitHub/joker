# Joker

Joker can help you run command in background, view running command list and stop it.

### Install

Download [joker](https://github.com/txthinking/mr2/releases/download/v20190810/joker)

### Source

    $ make
    $ make install

### Usage

    $ joker brook server -l :9999 -p password
    $ joker list
    $ joker stop <pid>
    $ joker log <pid>
